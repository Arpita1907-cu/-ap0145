First-time setup

In the project folder (where app.py lives):

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py

Then use http://127.0.0.1:8501/ or open web/index.html again.

Share on your network (optional)

streamlit run app.py --server.address 0.0.0.0 --server.port 8501

Others on the same Wi‑Fi: http://<your-ip>:8501

Preview static pages over HTTP (optional)

python3 -m http.server 8080

http://localhost:8080/web/index.html
